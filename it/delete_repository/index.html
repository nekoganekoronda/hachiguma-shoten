GitHubでリポジトリを削除する方法

こんにちは！八熊みゆです！
GitHubにリポジトリを作ったものの、ゴミすぎるソースをコミットしちゃったな～消したいな～ということはありませんか？
そんなときに、GitHubのリポジトリを削除する方法をご紹介します！

① GitHubを開く

まずは、GitHubを開きましょう。https://github.com/へアクセスして、ログインしてください。
ログイン後、右上にある自分のアイコン(赤枠で囲っている部分)をクリックしてください。

② リポジトリを開く

プルダウンの「Your repositories」をクリックしてください。

③ 削除するリポジトリを選択する

リポジトリの一覧画面が表示されるので、削除したいリポジトリをクリックしてください。

④ リポジトリのSettingsタブを開く

③で削除したいリポジトリをクリックすると、そのリポジトリの画面に遷移します。この画面で、Settingsタブ(赤枠部分)をクリックしてください。

⑤ Delete this repositoryをクリックする

リポジトリのSettingsタブを開いたら、一番下までスクロールしてください。すると、Danger Zoneというあからさまにヤバそうなゾーンが現れます。このDanger Zone内の「Delete this repository」と書かれた、これまたヤバそうなボタンをクリックしてください。

⑥ 確認画面に削除するリポジトリ名を入力する

Delete this repositoryボタンをクリックすると、確認画面が表示されます。テキストボックスに削除したいリポジトリ名を入力し、「I understand the consequences, delete this repository」と書かれた赤色のボタンをクリックしてください。

⑦ パスワードを入力する

パスワードの入力を求められます。テキストボックスにパスワードを入力し、「Confirm password」をクリック。

⑧ リポジトリが削除されたことを確認する１

ホーム画面に戻ります。①②と同様に、自分のアイコンをクリックし、「Your repositories」をクリック。

⑨ リポジトリが削除されたことを確認する２

リポジトリ一覧画面が表示されます。
先ほど操作したリポジトリが削除されていることを確認してください。
以上でリポジトリの削除は完了です。お疲れ様でした！
